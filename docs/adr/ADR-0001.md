# ADR-0001: Arquitectura elecgenflow

## Estado
Aceptado

## Decisión
Se debe establecer la adopción de Domain-Driven Design (DDD) y el patrón Model-View-Presenter (MVP), utilizando pandapower como motor de backend.

## Consecuencias
Se debe garantizar la estricta separación entre los modelos, la simulación y el reporting.
