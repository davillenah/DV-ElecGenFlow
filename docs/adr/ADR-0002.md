# ADR-0002: Reordenamiento del Roadmap por Dependencia de Capas Lógicas

## Estado
Aceptado (actualizado)

## Decisión
Se prioriza la integridad del modelo de datos antes de cálculos.
Se establece como dependencias obligatorias previas:
1) DSL de Tableros (Board DSL)
2) DSL de Red (Network DSL dirigida)
3) Registry / Component Library (tags + referencias técnicas)

## Consecuencias
- El motor evita carga manual duplicada (potencias/protecciones en niveles erróneos).
- La red se modela como DAG dirigido de alimentación.
- Simulación y reglas operan solo sobre entidades validadas y referenciadas.
