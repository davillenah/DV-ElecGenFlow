# CostModel (documental)

- La **EPIC-2 (Domain Core)** define únicamente los contratos y la estructura de datos para el modelo de costos, pero no incluye lógica de cálculo.
- La implementación funcional del modelo de costos (materiales, protecciones, cableado y penalizaciones por incumplimiento técnico) se ejecuta en la **EPIC-9 (Optimizer + Cost Model)**.
